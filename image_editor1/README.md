# image_editor

## 기능

- 갤러리에서 이미지 선택하기
- 스티커를 탭해서 이미지에 올리기
- 삭제 버튼 만들어서 추가한 이미지 삭제하기
- 저장 버튼 만들어서 수정한 이미지 갤러리에 추가하기

- 스티커 이미지 경로값
